⚙ Pipeline Architecture
⬇
Video Ingestion
Downloads YouTube ad → uploads to Azure Blob Storage for durable processing
🎬
Azure Video Indexer
Extracts speech transcript + OCR (on-screen text claims) from the video
🔍
Azure AI Search
Vector search over YouTube Ad Specs PDF + FTC Influencer Guide for relevant rules
🤖
GPT-4o Auditor
LangGraph node that reasons over transcript + rules to detect policy violations
📊
App Insights
Telemetry, tracing & structured logging for every pipeline run
📋
Compliance Report
Severity-graded violations with PASS/FAIL verdict + natural language summary
